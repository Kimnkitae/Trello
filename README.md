# Trello

## Как открыть сайт (самый простой способ)

1. Скачай архив с последней версией.
2. Распакуй архив в любую папку на компьютере.
3. Открой файл **`index.html`** двойным кликом — сайт сразу запустится в браузере.

Всё! Ничего устанавливать не нужно.
